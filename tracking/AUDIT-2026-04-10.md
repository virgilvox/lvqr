## Protocol Correctness Verification

The MoQ wire protocol implementation was verified by tracing every byte through the moq-lite Rust source code:

- **SETUP handshake**: CLIENT_SETUP (0x20) + u16 BE size + varint version codes. Verified with standalone Rust/JS test programs that produce identical bytes.
- **Path encoding**: confirmed `Path` is `Cow<str>`, encodes as `varint(len) + utf8`, NOT segmented.
- **Subscribe message**: varint id + string broadcast + string track + u8 priority. Confirmed u8 vs varint via moq-lite coding module.
- **Data streams**: varint DataType(0) + size-prefixed Group(subscribe_id, sequence) + frames(varint size + payload).
- **fMP4 boxes**: verified against W3C ISO BMFF spec. trun version 1 for signed CTS. esds descriptor lengths manually traced.
- **Varint roundtrip**: 0xff0dad01 encodes to C0 00 00 00 FF 0D AD 01, verified decode on both JS and Rust sides.

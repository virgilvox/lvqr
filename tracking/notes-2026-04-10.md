### Key Technical Decisions
1. moq-lite OriginProducer is the shared state - relay doesn't manually forward tracks
2. Publish/subscribe swap: relay publishes TO subscribers, consumes FROM publishers
3. RTMP ingest uses callback architecture (on_video, on_audio, on_publish, on_unpublish)
4. FLV keyframe detection via frame type nibble (upper nibble == 1)
5. Type aliases (MediaCallback, StreamCallback) to satisfy clippy complexity warnings
